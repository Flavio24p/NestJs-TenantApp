You Need:

- NPM
- Node
- NestJs
- Docker

Start Commands for docker-compose file:

- docker-compose up

To run in the background:

- docker-compose up -d

Start Commands for Dockerfile:

Build your image:

- docker build <your path> -t <<user>/project-name>

Run:

- docker run -p 5050:3000 <<user>/project-name>
